# 函数 

函数只有被调用才能被执行

## 匿名函数 函数字面量 
字面量其实就是值 {} -> 对象字面量
```
let test = function test1() {

}

test.name // test1;
```
```
function a(b){
    b =2
    console.log(arguments[0])
}
a(1)  // 2 这里的b已经被改成2了
```

系统会为函数默认添加一个return，终止函数执行


function test(a = 1, b){
    console.log(a)
    console.log(b)
}

test(undefined,2)  // 1,2
