# 小技巧

## 交换两个值

### 通过解构操作交换值
```
let a = 1;
let b = 2;
[a, b] = [b, a]
```
## 位运算

### 使用&运算符判断一个数的奇偶
```
// 偶数 & 1 = 0
// 奇数 & 1 = 1
console.log(2 & 1)    // 0
console.log(3 & 1)    // 1
```

### 使用~, >>, <<, >>>, |来向下取整
```
console.log(~~6.83)    // 6 当然是这个写法最简单
console.log(~~-6.83)    // -6
console.log(6.83 >> 0)  // 6
console.log(6.83 << 0)  // 6
console.log(6.83 | 0)   // 6
// >>>不可对负数取整
console.log(6.83 >>> 0)   // 6
```

### 使用^来完成值交换
```
var a = 5
var b = 8
a ^= b
b ^= a
a ^= b
console.log(a)   // 8
console.log(b)   // 5
```
## 数组的最后一个元素
```
const arr = ['justin1go', 'justin2go', 'justin3go'];
console.log(arr.at(-1)) // 倒数第一个值
console.log(arr.at(-2)) // 倒数第二个值
console.log(arr.at(0)) // 正数第一个  
console.log(arr.at(1)) // 正数第二个
```
